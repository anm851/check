# Ejercicio de Compresión y Encriptación

Este repositorio contiene un archivo `.rar` protegido con contraseña para que los usuarios practiquen la extracción y manipulación de archivos comprimidos usando Python. 

## Instrucciones
1. Usa el script `extract_rar.py` para intentar extraer el contenido del archivo comprimido.
2. La contraseña para el archivo `.rar` es `python123`.
3. Este ejercicio es únicamente con fines educativos, para aprender cómo trabajar con archivos comprimidos y encriptados en Python.

## Requisitos
- Instala el módulo `rarfile` en Python (requerido para manejar archivos `.rar`).

pip install rarfile
